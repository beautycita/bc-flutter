Experimental Physics

Electromagnetic Research

There are vast regions of electromagnetic variable space that have never been explored — not because they're impossible, but because nobody had a reason to try those specific combinations. These interactive models simulate configurations inspired by Tesla's lesser-known patents and modern materials science. Each one asks: what happens if we try something nobody has tried?

→

Counter-Rotating Bifilar Coil

Tesla's zero-inductance coil meets counter-rotating magnets. Explore how opposing magnetic fields interact with bifilar winding geometry.
