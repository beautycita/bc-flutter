2. Autenticación

Todas las solicitudes requieren autenticación HMAC-SHA256 mediante tres encabezados HTTP:

EncabezadoDescripciónEjemplo
X-SAT-KeyClave de API (identificador)9670aba0...6e29
X-SAT-TimestampTimestamp Unix en milisegundos1711648000000
X-SAT-SignatureFirma HMAC-SHA256a1b2c3d4...ef90

Cálculo de la Firma

// La firma se calcula así:
payload = timestamp + método_HTTP + ruta_completa
firma = HMAC-SHA256(secret, payload)

// Ejemplo:
timestamp = "1711648000000"
método    = "GET"
ruta      = "/sat-access/transactions?from=2026-01-01&to=2026-03-31"

payload   = "1711648000000GET/sat-access/transactions?from=2026-01-01&to=2026-03-31"
firma     = HMAC-SHA256(secret, payload)  → hex string
Ventana de tiempo: El timestamp debe estar dentro de un rango de 5 minutos del servidor. Solicitudes fuera de este rango serán rechazadas para prevenir ataques de repetición.

7. Ejemplo Completo (Python)

import hmac, hashlib, time, requests

API_KEY    = "9670aba0502a39a533d7835aa8047371ff782403d4a9200d26af4a571f986e29"
API_SECRET = "d9040c7de8530ee24f4eb9b71b49e77e643216a96c79a402f655aa895cb17fb0b70bf9d2d6842ec97188320b413d089477f5b35457e648ff5001f1e202220ff8"
BASE_URL   = "https://beautycita.com/supabase/functions/v1/sat-access"

def sat_request(path, params={}):
    url = f"{BASE_URL}/{path}"
    if params:
        qs = "&".join(f"{k}={v}" for k, v in params.items())
        url += f"?{qs}"

    timestamp = str(int(time.time() * 1000))
    sign_path = url.replace(BASE_URL.rsplit("/", 1)[0], "")
    payload = f"{timestamp}GET{sign_path}"

    signature = hmac.new(
        API_SECRET.encode(),
        payload.encode(),
        hashlib.sha256
    ).hexdigest()

    headers = {
        "X-SAT-Key": API_KEY,
        "X-SAT-Timestamp": timestamp,
        "X-SAT-Signature": signature,
    }

    return requests.get(url, headers=headers).json()

# Obtener transacciones de enero 2026
data = sat_request("transactions", {"from": "2026-01-01", "to": "2026-01-31"})
print(data)

# Obtener resumen mensual
summary = sat_request("summary", {"period": "2026-01"})
print(summary)

# Declaración de la plataforma
platform = sat_request("platform", {"period": "2026-01"})
print(platform)
